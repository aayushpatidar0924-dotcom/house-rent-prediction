# 🏠 House Rent Prediction using Machine Learning

## 📌 Project Overview

This project predicts house rent based on features like BHK, size, bathroom, location, and furnishing status using machine learning techniques.

---

## 🚀 Features

✔️ Data Cleaning & Preprocessing
✔️ Outlier Detection (IQR & Z-score)
✔️ Feature Engineering (Date transformation)
✔️ Encoding of categorical variables
✔️ Model Training using Linear Regression
✔️ Model Evaluation using R² Score

---

## 📊 Model Performance

**R² Score: 0.43**
🚧 Currently working on improving model performance
